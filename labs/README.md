# Agentic AI Examples

This package includes simple introductory LangChain agent examples for teaching a 60-minute session on agentic AI.

## Included Notebooks

### 1. `01_math_agent.ipynb`
A minimal agent that uses a calculator tool.

### 2. `02_search_agent.ipynb`
A simple agent using a custom fake search tool.

### 3. `03_csv_agent.ipynb`
An agent that loads and summarizes a CSV file.

## Requirements

```
pip install langchain langchain-openai pandas
```

## Usage

Open each notebook in Jupyter and run cells sequentially.
